# Windows Security Administration Lab

## Overview

This project documents a hands-on Windows 11 security and administration lab built using VMware Workstation. The goal of this lab was to create a virtualized Windows environment and practice foundational IT administration, networking, and security concepts.

## Lab Environment

- Operating System: Windows 11
- Virtualization Platform: VMware Workstation
- Network Configuration: NAT
- User Accounts:
- labadmin1 (Administrator)
- labuser (Standard User)

## Skills Demonstrated

- Virtual machine deployment and configuration
- Windows user account management
- Administrator vs standard user permissions
- Network configuration and connectivity testing
- Windows Firewall verification
- Endpoint security review
- Windows Update verification
- File and folder permissions (NTFS)
- Windows Event Viewer and security log review

## Security Concepts Practiced

- Least privilege
- Access control
- System hardening
- Patch management
- Security monitoring
- Troubleshooting
